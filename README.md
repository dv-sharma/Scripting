### VARIABLES 

- Storage location having name or name value pair
- Case sensitive
- No spaces before or after = sign
- Uppercase
- Syntax:
    VAR_NAME="value"

    e.g. SHELL_NAME="bash"

- use the $ sign to use the value of the variable

#### NOT VALID
- cannot start with number
- no special charactetrs
- only "_" is allowed
